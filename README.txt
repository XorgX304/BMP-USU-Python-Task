BMP README
==========

Данное риложение для предназаначено для просмотра внутренней структуры файла и его отображения.

Приложение состоит из двух частей:
1) bmp_module.py содержит все необходимые структуры для работы с файлами формата bmp.
2) bmp_gui.py содержит описание удобного пользовательсткого интерфейса.

Описание тестов:
input00.bmp - маленкое изображение
input01.bmp - большое изображение
input02.bmp - среднее изображене
input03.bmp - среднее изображение (прямой порядок строк, т.е. height < 0)
input04.bmp - среднее изображене (32 бит)
input05.bmp - среднее изображене (16 бит)
input06.bmp - 4-бит изображение
input07.bmp - 8-бит изображение
input08.bmp - 1-бит изображение
input09.bmp - 1x1 изображение
